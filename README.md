Developed a full-stack booking application using the MERN stack and TypeScript, enabling users to register,
authenticate, and manage bookings with robust security practices.
• Implemented JWT-based authentication with HTTP-only cookies for secure session management, mitigating
XSS and CSRF vulnerabilities.
• Built a responsive and user-friendly frontend using React and TypeScript, consuming RESTful APIs for
efficient client-server communication.
• Integrated Stripe for secure payment processing and Cloudinary for optimized image uploads and storage.
